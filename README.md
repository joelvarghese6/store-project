# Ecommerce Assignment

A responsive product dashboard built using Next.js, React, and Tailwind
CSS, featuring product listing, sorting, filtering, searching, and cart
functionality.

### Core Features

-   Product listing with image, name, category, price, rating, and "Add
    to Cart" button
-   Sorting by:
    -   Price: Low to High / High to Low
    -   Rating
-   Filtering by category (3+ categories)
-   Search by product name
-   Cart system with add/remove functionality
-   Cart item counter in header
-   Fully responsive grid:
    -   1 column on mobile
    -   2 columns on tablets
    -   3--4 columns on desktop
